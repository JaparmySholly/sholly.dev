# Cyber Portfolio Starter

Modern cybersecurity + software engineering portfolio starter.

## Stack
- Next.js 15
- TypeScript
- Tailwind CSS
- Supabase
- PostHog
- Sentry
- Resend
- Cloudflare-ready
- Vercel-ready

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Rename `.env.example` to `.env.local`

## Deploy
- Push to GitHub
- Connect to Vercel
- Configure Cloudflare DNS
