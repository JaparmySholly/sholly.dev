# UI Components Setup Instructions

Due to environment constraints, please execute the following command in your terminal to create the UI components:

```bash
node setup-ui.js
```

This script has been created at:
`c:\Users\HP\Desktop\portfolio-starter\setup-ui.js`

Alternatively, if you prefer a manual approach, you can:

1. Create the directory: `mkdir -p src/components/ui`
2. Run: `bash setup-ui.sh`

Or manually create each file in `src/components/ui/`:
- Button.tsx
- Card.tsx
- Badge.tsx
- AnimatedSection.tsx
- Navbar.tsx
- index.ts

The setup scripts contain all the necessary code.
